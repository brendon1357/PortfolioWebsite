Space Invaders Inspired Remake

Application Information

This game is a Space Invaders inspired game made using Python and the Python game library Pygame. The game incorporates key programming fundamentals such as conditional statements, loops, arrays, mathematical expressions, and boolean variables. The game has a score tracker which increments difficulty based on the player's score. As the player, you control a spaceship using the left and right arrow keys to move left and right respectively, and spacebar to shoot at the enemies. The enemies slowly progress closer towards you until they cross the boundary marked by a red line in which your life score is decremented by 1 point from a maximum of 3. If you reach 0 lives the game ends and you lose, or if you reach the maximum score of 100 points the game ends and you win. I've included a link to the source code and a link to download the executable file for the game so you can play it yourself.

How To Run

First download the game file by selecting the "Download Game" link and then run the installer. For simplicity sake, it would be best to add the directory of the installation to your desktop. Once the installation has finished open up the folder titled "FinishedSpaceInvadersGame" in the location you chose to install it. You will see all the game dependencies and two files with the name "main". Double click the application type file titled "main" and this will run the game for you. Since I do not have any digital signatures for this game, windows defender and other antivirus softwares may see the file as a threat since it is an executable file. The download is only compatible with windows. If windows defender sends you a warning message upon installation about the file simply click "More info" on the warning screen and select "Run anyway". A similar process would most likely apply to other antivirus softwares.
