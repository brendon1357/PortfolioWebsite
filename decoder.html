Password Decoder

Application Information

This application was made using Python and Tkinter. The application works by allowing a user to register into a database, log into the application and decode/encode passwords. Once logged in, the user must press the "Generate Password" button to generate a randomized password. Once generating the password the user must press the "Encode Password" button to encode the generated password. The user then has to store this encoded password in an easy to retrieve, safe location. When logging in again the user can now paste the encoded password into the textbox next to the "Decode Password" button, then press the "Decode Password" button to retrieve the original generated password. This works by having the program generate a unique key that can be used to encode/decode as many passwords as the user would like to. The idea is to add an extra layer of security to help protect the user's passwords. There is no limit on how many passwords can be encoded and decoded with a single key.

How To Run

Selecting the "Download App" button below will automatically start the installation of the executable file for the application. Once the download has finished, just simply start the application and it will run. The download is only compatible with windows. If windows defender sends you a warning message upon installation about the file simply click "More info" on the warning screen and select "Run anyway". A similar process would most likely apply to other antivirus softwares. A warning is displayed because the executable does not have a recognized digital signature.
